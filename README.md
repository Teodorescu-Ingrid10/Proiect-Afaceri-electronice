# Proiect-Afaceri-electronice
Aplicatia simuleaza o platforma de gestionare a produselor in care utilizatorii isi 
creeaza cont, se autentifica si pot adauga produse intr-o lista personala de favorite (Wishlist). 
Utilizatorii pot vizualiza produsele disponibile si, daca au rolul de admini, pot adauga si edita 
produse.
